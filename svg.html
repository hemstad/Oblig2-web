Hva er SVG og hva bruker man SVG til?

Hva er SVG?

SVG er grafikk tegning. Det kan minne oss om noen elementer fra Canvas, og kan på mange måter se litt likt ut, men dette er en litt annen type grafikk. SVG kan være både tekst og bilde.

Hva brukes SVG til?

Det er mange grunner til å bruke SVG. Hovedsakelig brukes SVG til å definere grafikk for internett. Det er lettere å skalere de, de bruker mindre plass enn for eksempel JPG og PNG-bilder, man kan bruke det samme bilde flere ganger på en side, og man kan endre de ved hjelp av både Javascript og CSS.
